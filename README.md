# Revtek — Future-Proof Digital Solutions

![Revtek Preview](https://via.placeholder.com/1400x800/0a0a0a/00f2ff?text=Revtek+Modern+Landing+Page)

A **single-page, ultra-modern, high-conversion** landing + contact website built for Revtek — a forward-thinking digital studio.

Live Demo: [https://revtek.pages.dev] 

## Features

- Stunning futuristic design with glassmorphism & neon cyan accents  
- Animated particle background (lightweight, mouse-reactive)  
- Fully responsive — looks perfect on mobile, tablet, and desktop  
- Real working contact form (powered by Formspree)  
- Smooth micro-interactions and hover effects  
- SEO-optimized with proper meta tags  
- No external dependencies — 100% vanilla HTML/CSS/JS  
- Fast loading, modern typography, and premium feel  

## Tech Stack

- HTML5
- CSS3 (Custom properties, Grid, Flexbox, Backdrop-filter)
- Vanilla JavaScript (ES6+)
- [Formspree](https://formspree.io) for contact form (free tier available)
